## Why API Dev Studio?

- **Local-first** — Everything runs on your machine. No cloud, no accounts, no data sharing.
- **All-in-one** — Replace webhook.site, Postman mocks, and json-server with a single app.
- **Fast** — Native desktop app built with Rust. Starts instantly, runs efficiently.
- **Developer-friendly** — Template variables, conditional responses, request inspector, and more.
