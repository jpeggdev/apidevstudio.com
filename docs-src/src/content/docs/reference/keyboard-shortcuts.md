# Keyboard Shortcuts

API Dev Studio supports keyboard shortcuts for common actions. Use `Ctrl` on Windows/Linux or `Cmd` on macOS.

## Global Shortcuts

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Open Command Palette | `Ctrl+K` | `Cmd+K` |
| Toggle Help Panel | `F1` | `F1` |
| Show Keyboard Shortcuts | `Ctrl+/` | `Cmd+/` |
| Undo | `Ctrl+Z` | `Cmd+Z` |
| Redo | `Ctrl+Y` or `Ctrl+Shift+Z` | `Cmd+Shift+Z` |

## Navigation

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Go to Projects | `Ctrl+1` | `Cmd+1` |
| Go to Endpoints | `Ctrl+2` | `Cmd+2` |
| Go to Requests | `Ctrl+3` | `Cmd+3` |
| Go to Settings | `Ctrl+,` | `Cmd+,` |

## Project Actions

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| New Project | `Ctrl+N` | `Cmd+N` |
| Start/Stop Server | `Ctrl+Enter` | `Cmd+Enter` |

## Endpoint Actions

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| New Endpoint | `Ctrl+Shift+N` | `Cmd+Shift+N` |
| Save Endpoint | `Ctrl+S` | `Cmd+S` |
| Duplicate Endpoint | `Ctrl+D` | `Cmd+D` |
| Delete Endpoint | `Delete` | `Backspace` |

## Request Inspector

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Refresh Requests | `Ctrl+R` | `Cmd+R` |
| Star Request | `S` | `S` |
| Clear All Requests | `Ctrl+Shift+Delete` | `Cmd+Shift+Backspace` |

## Editor

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Format JSON | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| Toggle Comment | `Ctrl+/` | `Cmd+/` |
| Find | `Ctrl+F` | `Cmd+F` |

## Command Palette

The Command Palette (`Ctrl+K` / `Cmd+K`) provides quick access to all actions. Start typing to filter:

- `>new` - Create new project or endpoint
- `>start` - Start server
- `>stop` - Stop server
- `>import` - Import OpenAPI or Postman
- `>export` - Export project
- `>settings` - Open settings

## Customizing Shortcuts

Custom keyboard shortcuts are not yet supported but planned for a future release.
